# Huaben and Recording Rules

> Version: v0.6.9
> Status: 正式发布，持续迭代

## 1. 分支产物

| 产物 | 定位 | 规则 |
|---|---|---|
| 分集文本 | 集数规划后的生产底稿 | 保留原文范围、起止句、标题、时长参数和切集依据 |
| 原文画本 | 忠于原文的结构标注母版 | 可由分集文本直接生成；完成后可停在当前产物，也可继续进入录制执行稿或录制版；不绑定 CV，不做最终排版，可引用角色色标 |
| 录制执行稿 | 可录、可演、可剪、可制作的声音执行稿 | 默认服务有声书 / 多人有声书模式；可由分集文本或原文画本派生；承接听觉友好改写、声音单元拆分、参考语气、音效、配乐和待确认项；必须与其他产物物理隔离，改动可回溯 |
| 录制版画本 | 给录制者 / 制作人使用的最终呈现版 | 由原文画本或录制执行稿派生；显示颜色、按需 CV / 录制责任、兼任提示、补录与校准区 |

## 2. 原文画本核心标签

- `【前情提要】`
- `【正文开始】`
- `【旁白】`
- `【角色名（情绪/状态）】`
- `【角色名｜心声（情绪/状态）】`
- `【系统音】`
- `【群体声｜处理方式（状态）】`
- `【特殊声具体类型】`
- `【场景：...】`
- `【音效：...】`
- `【引用】`
- `【旁白｜回忆】`

## 3. 标注规则

- 旁白、对白、心声、系统音、群体声必须区分。
- 场景 / 音效可独立成行，不能脑补未出现或不必要的音效。
- 群体声要标注处理方式：多人同录 / 单人兼任 / 后期叠加 / 音效替代 / 待定。
- 前情提要进入画本时，必须用 `【前情提要】`，并在正文前加 `【正文开始】`。
- 原文画本不写 CV 名，不做最终字体、颜色、页眉页脚排版。

## 4. 题材扩展标签

按需启用，不默认塞满：

- 系统流：任务、面板、奖励、惩罚、机制提示；
- 书信 / 日记：书信、日记、文书、档案；
- 聊天 / 论坛：群聊、弹幕、论坛帖、私信；
- 古风 / 诗词：诏书、诗词、唱词、祭文；
- 叙事结构：回忆、梦境、旁白插叙、多视角切换。

## 5. 角色颜色规则

- 颜色绑定角色，不绑定 CV。
- 主角色和高频角色优先独立色。
- 低频角色可共用色。
- 独立颜色数量建议控制在 8–12 个以内。
- 原文画本可引用色标，录制版画本负责最终呈现。

## 6. 录制执行稿

录制执行稿不是原文画本之后的固定步骤，原文画本也不是录制执行稿之前的固定步骤。集数规划完成后，AI 应根据项目要求选择下一步动作：

- 先原文画本：分集文本 → 原文画本，之后可停在当前产物、进入录制执行稿或进入录制版；
- 先录制执行稿：分集文本 → 录制执行稿，之后可停在当前产物或进入录制版；
- 承接已有产物：如果已经有原文画本或录制执行稿，继续做下一产物时，应把已有产物作为可回查输入之一。

路径选择：

- 只需忠于原文结构标注：原文画本；
- 需要听觉友好、角色拆分、短旁白转 CV、文字载体听感判断、参考语气、音效或配乐：录制执行稿；
- 只需最终给录制者使用，且输入已足够：录制版画本；
- 不需要：跳过对应层。

声音产物形态边界：

- 有声书 / 多人有声书：使用当前默认录制执行稿规则。
- 轻广播剧化：可在录制执行稿内扩大声音化尺度，但不改成广播剧剧本。
- 广播剧 / 声音剧：不由当前录制执行稿规则默认生成。

禁止：

- 污染原文整理、分集文本或原文画本；
- 无依据加戏；
- 改变主线剧情；
- 把探索样张直接当长期规则。

## 7. 录制版画本

录制版由原文画本或录制执行稿派生。必须按需呈现：

- 角色颜色；
- 录制责任人 / CV；
- 前情提要；
- 同场景兼任提示；
- 待确认 / 补录记录；
- 试录 / 校准回填。

## 8. CV 兼任提示

原文画本不处理 CV 兼任。录制版画本阶段，同一 CV 兼任多个角色且同场景直接对话时，在场景前集中提醒：

```text
【兼任提示】本场{CV名}兼任：{角色A}、{角色B}。两人有直接对话，建议分角色录制，注意声线区分。
```

可选处理方式：同场连续录但换声线、分角色分段录、后期拼接、重新分配 CV。

## 9. 角色工作稿

不是默认产物。仅在复杂协作、主角训练、临时加入、补录、录制方明确要求时派生。
