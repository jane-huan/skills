# Artifact Schemas

> Version: v0.6.9
> Status: 正式发布，持续迭代

本文件定义前期筹备产物的最小 schema。字段说明保持够用即可；解释性说明不要塞进这里。

---

## 1. 项目初始化记录

### 用途
为阅读、切集、原文画本、录制执行稿、录制版和时长校准提供约束。

### 使用原则
新书 / 新项目启动时由 AI 主动建立轻量记录。它不是一次性填完的大问卷；已知信息先落地，未知信息标“待确认”，后续聊天中有新决定时滚动回填。

### 字段

| 字段 | 说明 |
|---|---|
| 书名 | 项目名称 |
| 类型 / 题材 | 纯有声书 / 多人演播 / 轻广播剧化等 |
| 目标平台 | 平台或发行场景 |
| 单集目标成品时长 | 分钟或秒 |
| 片头时长 | 固定值或无 |
| 是否需要前情提要 | 是 / 否 / 待定 |
| 前情提要模式 | 固定 / 每集变量 / 无 |
| 前情提要目标时长 | 分钟或秒 |
| 前情提要是否进入录制文本 | 是 / 否 / 待定 |
| 片尾时长 | 固定值或无 |
| 正文目标时长 | 成品时长扣除片头、前情提要、片尾 |
| 初始朗读速度 | 默认 220–280 字/分钟 |
| 是否试录校准 | 是 / 否 |
| 声音产物形态 | 有声书 / 多人有声书 / 轻广播剧化 / 广播剧或声音剧 / 待确认 |
| 声音处理强度 | 不处理 / 原文画本 / 录制执行稿 / 轻广播剧化 |
| 下一步生产动作 | 先原文画本 / 先录制执行稿 / 进入录制版 / 停在当前产物 / 待确认 |
| CV / 录制责任需求 | 正式分配 / 轻量责任 / 待定 |
| 是否需要彩色录制版 | 是 / 否 |
| 平台规则 / 审核禁忌 | 已知规则或待确认 |
| 合同要求 | 已知要求或待确认 |
| 当前状态 | 已确认 / 待确认 / 建议默认值 |
| 回填记录 | 后续确认或修改的日期与内容 |
| 制作人备注 | 补充约束 |

## 2. 阅读理解块记录

### 用途
为资料包、集数规划和后续画本提供可回查理解过程。

### 字段

| 字段 | 说明 |
|---|---|
| 块编号 | 如 RB-001 |
| 输入形态 | 已分集 / 未分集 |
| 覆盖范围 | 章节、段落或集数 |
| 字数 | 本块字数 |
| 起始句 | 精确定位 |
| 结束句 | 精确定位 |
| 主要内容 | 3–6 条剧情推进 |
| 主要人物 | 实际出现或重点提及的人物 |
| 新增重要信息 | 设定、伏笔、关系变化、重要物件、关键事件 |
| 可进入全书资料包的信息 | 后续可复用的信息 |
| 潜在切集点 | 自然切点、悬念点、转场点 |
| 切集点理由 | 章首入口、章尾停顿、语义完整性说明 |
| 需保护语义块 | 对话、短信、电话、语音往返、信件、日记、记录、清单、连续编号、分项展开、作品结构、象征线索、争执、动作链、观点链、情绪链、行动闭环等 |
| 原文问题 | 错字、缺文、归属不明、逻辑疑点 |
| 是否更新资料包草稿 | 是 / 否 / 待定 |
| 制作人备注 | 人工判断 |

## 3. 全书资料包

### 用途
作品理解手册，服务制作人、AI、CV 和后续原文画本、录制执行稿或录制版生产。

### 章节

| 章节 | 内容 |
|---|---|
| 作品概览 | 书名、题材、篇幅、预计集数、核心卖点、整体风格、目标听感、一句话简介 |
| 全书故事梗概 | 一句话梗概、详细梗概、主线推进、阶段节点 |
| 世界观 / 基础设定 | 背景、规则、地点、组织、特殊设定、不可读错或改错的设定 |
| A 类角色画像 | 定位、年龄感、身份、气质、性格、目标、情绪底色、说话方式、关系、声音建议、弧光、注意事项 |
| B 类角色画像 | 定位、关系、性格、说话方式、声音建议、出场阶段、是否独立声线、注意事项 |
| C 类角色画像 | 角色名、场景、剧情功能、一句话声音提示、是否可群体声 / 兼任 |
| 角色分析工作簿 | 全书汇总阅读理解后生成；包含角色拆解、人物画像、剧情对照、台词统计、选角与兼任建议 |
| 人物关系 | 核心关系、阵营、情感、冲突、关系变化 |
| 演播风格总说明 | 旁白、对白尺度、系统音 / 特殊音、情绪尺度、节奏要求 |
| 统一规则与注意事项 | 统一称呼、易读错词、不能改动的设定、审核风险、制作人要求 |
| 前情提要规则 | 是否需要、风格、固定 / 变量、避免剧透规则 |
| 原文问题与修订回流 | 重要问题摘要、影响范围、处理状态 |

## 3.1 角色分析工作簿

### 用途
在 AI 完成阅读理解块、汇总全书阅读理解并形成全书资料包 v1 后生成，用于制作人判断角色重要度、声音方向、CV 选角、兼任风险、录制排期和后续角色颜色与录制责任记录。

### 生成条件
- 已完成全书或当前生产范围的阅读理解块记录。
- 已完成统一汇总的全书阅读理解。
- 角色、剧情、出场范围、台词统计均能回查到阅读块、分集文本、原文画本、录制执行稿或原文统计依据。
- 不确定信息必须标“待确认”，不要用都市青年、成熟男性等泛化词替代可判断的年龄区间和具体身份。

### 工作簿建议表

| 表名 | 用途 |
|---|---|
| 01_角色总表 | 角色拆解、人物画像、声音建议和制作优先级的主表 |
| 02_剧情对照 | 角色在全书阶段、集数和关键事件中的功能变化 |
| 03_台词统计 | 出场顺序、出现篇幅、台词段数、台词字数、原文提及次数等可量化信息 |
| 04_选角兼任 | 独立声线、可否兼任、同场景风险、试音优先级和 CV 条件 |
| 05_待确认项 | 年龄、身份、关系、台词归属、声线尺度、兼任风险等待制作人确认内容 |

### 01_角色总表字段

| 字段 | 说明 |
|---|---|
| 角色名 | 统一称呼，含别名时另列 |
| 别名 / 称呼 | 原文中出现的其他称呼 |
| 角色档位 | A / B / C |
| 角色类型 | 主角 / 重要配角 / 功能配角 / 反派 / 龙套 / 群体 / 旁白 / 系统音等 |
| 性别 | 男 / 女 / 不限 / 待确认 |
| 年龄区间 | 如 18-22 岁、30-35 岁；无法确定时写年龄感 + 待确认 |
| 身份 / 职业 / 阵营 | 角色在故事中的社会身份和阵营 |
| 人物定位 | 在主线中的功能，不只写身份 |
| 人物经历简介 | 以本书完整故事为单位，简短概括角色在故事里的主要经历和处境；不写无关生平流水账 |
| 核心目标 / 欲望 | 角色主要想要什么 |
| 性格 / 气质 | 稳定人格特征和外显气质 |
| 情绪底色 | 长期情绪基调，如压抑、戒备、轻佻、冷硬 |
| 说话方式 | 句式、语速、口头禅、礼貌度、攻击性、信息密度 |
| 关系核心 | 与主角、反派、亲友、组织的关键关系 |
| 人物弧光 / 变化 | 从开场到结尾的主要变化 |
| 剧情功能 | 推动案件、制造冲突、提供线索、情感托举、世界观说明等 |
| 声音建议 | 年龄感、音色、节奏、情绪尺度 |
| CV 声线条件 | 试音或选角时必须具备的能力 |
| 演播注意事项 | 易过火、易脸谱化、需克制、需区分阶段等 |
| 颜色建议 | 是否建议独立颜色、共用色或待定 |
| 制作优先级 | 高 / 中 / 低 |
| 证据来源 | 阅读块编号、集号、章节或统计表引用 |
| 待确认项 | 制作人需要确认的信息 |

### 02_剧情对照字段

| 字段 | 说明 |
|---|---|
| 角色名 | 统一称呼 |
| 角色档位 | A / B / C |
| 首次出场 | 集号 / 章节 / 原文位置 |
| 最后出场 | 集号 / 章节 / 原文位置 |
| 出场顺序 | 按首次出场排序 |
| 覆盖集数 / 篇幅 | 如第01-94集（共87集） |
| 关键阶段 | 开端 / 发展 / 转折 / 高潮 / 结局，或项目自定义阶段 |
| 关键事件 | 与角色有关的剧情节点 |
| 关系变化 | 与其他角色的关系推进 |
| 情绪变化 | 本阶段主要状态 |
| 声音阶段提示 | 本阶段声音和表演差异 |
| 关联阅读块 | 可回查来源 |

### 03_台词统计字段

| 字段 | 说明 |
|---|---|
| 角色名 | 统一称呼 |
| 角色档位 | A / B / C |
| 出场顺序 | 首次出场排序 |
| 首次出场 | 集号 / 章节 |
| 出现的篇幅 | 集号范围和出现集数 |
| 出现集数 | 数值 |
| 台词段数 | 可按引号台词、画本角色标签或人工校对口径统计 |
| 台词字数 | 角色台词总字数 |
| 原文提及次数 | 辅助判断存在感，不等同于台词量 |
| 旁白承载量 | 对旁白或无对白重要角色可记录高 / 中 / 低 |
| 统计口径 | 自动统计 / 画本统计 / 人工校对 |
| 统计来源 | 原文、原文画本、录制执行稿、脚本或工具输出 |
| 置信度 | 高 / 中 / 低 |
| 统计备注 | 未明确归属、群杂台词、旁白引用等说明 |

### 04_选角兼任字段

| 字段 | 说明 |
|---|---|
| 角色名 | 统一称呼 |
| 角色档位 | A / B / C |
| 是否独立声线 | 是 / 否 / 待定 |
| 是否建议单独录制 | 是 / 否 / 待定 |
| CV 声线条件 | 试音硬条件 |
| 可否兼任 | 是 / 否 / 待定 |
| 建议兼任方向 | 可与哪些低频角色同 CV，或不建议兼任 |
| 同场景兼任风险 | 高 / 中 / 低 / 待查 |
| 风险依据 | 同集同场景直接对话、情绪冲突、声线相近等 |
| 试音优先级 | 高 / 中 / 低 |
| 试音片段建议 | 集号 / 场景 / 情绪类型 |
| 录制责任人 / CV | 可为空，后补 |
| 备注 | 制作人补充 |

### 05_待确认项字段

| 字段 | 说明 |
|---|---|
| 项目 | 角色 / 统计 / 声线 / 兼任 / 剧情 |
| 角色名 | 涉及角色 |
| 问题 | 需确认内容 |
| 影响产物 | 资料包 / 角色颜色 / 录制责任 / 原文画本 / 录制执行稿 / 录制版画本 |
| 建议处理 | AI 建议或待制作人判断 |
| 状态 | 待确认 / 已确认 / 暂不处理 |
| 备注 | 制作人补充 |

### 表格排版建议

| 元素 | 建议 |
|---|---|
| 冻结窗格 | 冻结首行；角色总表可冻结前 2-3 列 |
| 筛选 | 所有表开启筛选 |
| 颜色 | A 类深色、B 类中性色、C 类浅色；待确认项用警示色 |
| 条件格式 | 试音优先级、同场景兼任风险、置信度、制作优先级可用颜色区分 |
| 列宽 | 长文本列自动换行，统计列窄列居中 |
| 数据校验 | 角色档位、角色类型、风险等级、状态使用下拉选项 |

## 4. 集数规划记录

### 用途
控制每集范围、时长、钩子和后续画本生产状态。

### 每集字段

| 字段 | 说明 |
|---|---|
| 集号 | 第 001 集 |
| 集标题 | 可后续调整 |
| 原文范围 | 起止章节 / 段落 |
| 起始句 | 精确定位 |
| 结束句 | 精确定位 |
| 正文字数 | 不含片头、前情提要、片尾 |
| 核心事件 | 3–5 条 |
| 结尾钩子 | 听觉钩子或无钩子说明 |
| 章首独立性检查 | 可独立进入 / 依赖上一集 / 需调整 |
| 章首风险 | 是否属于上一场景的回复、反应、解释、心理落点或连续材料中段 |
| 章尾完整性检查 | 已收住语义块 / 留有未收束内容 / 需调整 |
| 被保护语义块 | 本集保留的完整对话、短信、电话、语音往返、信件、日记、记录、清单、连续编号、分项展开、作品结构、象征线索、争执、动作链、观点链、情绪链、行动闭环等 |
| 整体表达单元检查 | 连续编号 / 分项展开 / 清单记录 / 作品结构 / 象征线索 / 行动闭环是否完整 |
| 语义切断风险 | 无 / 轻微 / 需制作人确认 / 必须调整 |
| 标题贴合度 | 概括本集核心推进 / 偏局部噱头 / 待调整 |
| 标题吸引力检查 | 关系真相 / 情绪刺点 / 认知反转 / 人物觉醒 / 选择代价 / 核心矛盾 / 判断发现 / 待调整 |
| 标题风险 | 剧情复述 / 内容错位 / 剧透过度 / 营销腔 / 脱离剧情 / 无 |
| 单集目标成品时长 | 目标时长 |
| 片头时长 | 固定或无 |
| 前情提要模式 | 固定 / 每集变量 / 无 |
| 前情提要目标时长 | 目标时长 |
| 片尾时长 | 固定或无 |
| 正文目标时长 | 目标成品时长扣除片头、前情提要、片尾 |
| 当前朗读速度 | 初始估算 / 校准值 |
| 正文预计时长 | 正文字数 ÷ 当前朗读速度 |
| 成品预计时长 | 片头 + 前情提要 + 正文 + 片尾 |
| 时长偏差等级 | 正常 / 标记 / 需确认 / 必须处理 |
| 下一步生产动作 | 先原文画本 / 先录制执行稿 / 进入录制版 / 停在当前产物 / 待确认 |
| 原文画本状态 | 未开始 / 生成中 / 待校对 / 已确认 |
| 录制执行稿状态 | 不需要 / 待判断 / 生成中 / 已确认 |
| 录制版状态 | 未生成 / 待确认 / 待录制 / 已录制 |
| 备注 | 制作人补充 |

## 6. 原文画本

### 用途
逐集结构标注母版，服务校对、录制版派生和后续 AI 处理；可直接由分集文本生成；完成后可停在当前画本，也可继续进入录制执行稿。

### 章节

| 章节 | 内容 |
|---|---|
| 本集基础信息 | 书名、集号、范围、字数、时长、版本、状态 |
| 本集制作提示 | 核心事件、听感目标、主要角色 |
| 前情提要设置 | 无 / 固定 / 每集变量，是否进入录制文本 |
| 原文画本正文 | 统一标签标注的正文 |
| 本集待确认问题 | 位置、问题、建议处理、状态 |
| 派生说明 | 生成录制版时才加入颜色、CV、排版、兼任提示 |
| 试录 / 校准回填区 | 后续回填实际时长和校准速度 |

### 规则
忠于原文；不改剧情；不加戏；不主动做听觉友好改写；不默认加入参考语气、配乐设计或复杂后期提示；不绑定 CV；不做最终排版；可引用角色色标。

## 7. 录制执行稿

### 用途
分集后的可选生产节点，默认服务有声书 / 多人有声书模式，用于听觉友好、声音单元拆分、表演提示、音效配乐提示、轻广播剧化扩展或平台上传文本；可作为本轮共创终点，也可继续派生录制版画本；也可承接已有原文画本继续生成。广播剧 / 声音剧不由本规则默认生成。

### 字段 / 章节

| 字段 | 说明 |
|---|---|
| 输入底稿 | 分集文本 / 原文画本 / 其他可回查文本 |
