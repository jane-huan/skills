# QA and Calibration

> Version: v0.6.9
> Status: 正式发布，持续迭代

## 1. 前情提要规则

前情提要同时可能是：

1. 时长参数；
2. 录制文本。

必须确认：是否需要、固定 / 每集变量、目标时长、是否进入录制文本。进入画本时使用：

```text
【前情提要】……
【正文开始】
```

## 2. 时长公式

```text
成品预计时长 = 片头时长 + 前情提要时长 + 正文预计时长 + 片尾时长
正文目标时长 = 单集目标成品时长 - 片头时长 - 前情提要时长 - 片尾时长
正文预计时长 = 正文字数 ÷ 当前采用朗读速度
实际正文时长 = 实际成品时长 - 实际片头时长 - 实际前情提要时长 - 实际片尾时长
校准朗读速度 = 正文字数 ÷ 实际正文时长
```

开工前用 220–280 字/分钟粗估；第 1 集或试录后回填校准值。

## 3. 偏差阈值

| 偏差 | 判断 |
|---|---|
| ±10% 内 | 正常 |
| ±10%～15% | 标记 |
| ±15%～25% | 需制作人确认 |
| > ±25% | 必须重新评估切集 |

连续 3 集超过 ±15%、超过 30% 集数超过 ±15%、任一集超过 ±25%，视为系统性超标信号。

## 4. QA 节点最低通过标准

### 4.1 项目初始化

- 单集目标成品时长已确认。
- 片头 / 前情提要 / 片尾时长已确认或明确为无。
- 前情提要是否进入录制文本已确认。
- 初始朗读速度已给出或采用 220–280 字/分钟估算。
- 声音产物形态已确认或标为待判断。
- 声音处理强度已确认或标为待判断。
- CV / 录制责任需求已确认或标为待定。
- 平台规则、审核禁忌、合同要求未知时已标为待确认。

### 4.2 阅读理解块

- 覆盖范围、起始句、结束句明确。
- 主要内容为 3–6 条。
- 主要人物只记录实际出现或重点提及者。
- 新增设定、伏笔、关系变化、重要物件有记录。
- 可进入资料包的信息单独列出。
- 潜在切集点有章首 / 章尾理由。
- 不宜切断的完整语义块有记录。
- 连续编号、分项展开、清单式记录、阶段性总结、作品结构拆解、象征线索或行动闭环等整体表达单元有记录。
- 未脑补后续剧情。

### 4.3 全书资料包

- 作品概览完整。
- 主线梗概和阶段节点清楚。
- A/B/C 角色颗粒度区分。
- 重要设定、统一称呼、易错词已记录。
- 演播风格说明可指导画本。
- 原文问题和待确认项有回流入口。

### 4.4 集数规划

- 每集原文范围、起始句、结束句明确。
- 正文字数有记录。
- 片头、前情提要、片尾进入公式。
- 正文目标时长和预计成品时长已计算。
- 每集有结尾钩子或无钩子说明。
- 时长偏差等级可判断。
- 已逐集复审章首是否能独立进入。
- 章首未落在上一场景的直接回复、反应、解释、心理落点或连续材料中段。
- 章尾收住当前语义块，停在可自然暂停的位置。
- 未硬切完整对话、短信、电话、信件、清单、争执、谈判、动作链、观点链或情绪链。
- 未硬切连续编号、分项展开、作品结构、象征线索、阶段性总结或行动闭环。
- 未把“事件发生—角色回应—心理理解”拆成两集；如必须拆分，已给出制作人确认项。
- 标题对应本集核心推进，不只抓局部噱头。
- 标题不是直白剧情复述，不和上一集或下一集内容错位。
- 标题未过度剧透关键反转，未为了金句感脱离剧情。
- 标题对未听过正文的听众也有判断、疑问或情绪牵引。

### 4.5 角色颜色与录制责任

- A/B/C 角色分层合理。
- 颜色绑定角色，不绑定 CV。
- 主角色 / 高频角色有独立色或明确待定。
- 低频角色共用色有说明。
- 录制责任可为空，但状态明确。
- 同场景兼任风险有标记或待查。

### 4.6 原文画本

- 忠于原文，不改剧情。
- 旁白、角色、心声、系统音、群体声区分清楚。
- 前情提要如进入正文，使用 `【前情提要】`。
- 不主动做听觉友好改写，不默认加入参考语气、配乐设计或复杂后期提示。
- 不写 CV。
- 不做最终排版。
- 待确认问题有记录。

### 4.7 录制执行稿

- 是否明确由分集文本、原文画本或其他输入底稿派生。
- 声音产物形态明确；广播剧 / 声音剧未误用当前默认录制执行稿规则。
- 执行目标明确：听觉友好、可录、可演、可剪或轻广播剧化。
- 承接方式明确：夹叙信息先按听觉连续性判断，状态、动作、空间、心理、文字、转场和情绪底色没有丢失。
- 声源归属不易误解：能区分对白、心声、旁白判断、文字内容和现场反应。
- 没有为了声音化而过度使用群体声、特殊声、心声或多声部。
- 连续旁白和记录性文字没有因机械分段被切碎。
- 没有为了顺耳删掉原文中的状态、动作、心理或关系反应。
- 没有为了忠实原文把所有夹叙信息拆成零碎旁白。
- 完整叙述段没有因为含有可演状态而被拆碎。
- 纯视觉动作或神态没有被直接塞进 CV 台词括号。
- 配乐提示服务后期执行，必要时有进入点、持续范围、淡出位置、音量关系或素材检索方向。
- 转场提示服务场景进入，来自原文和上下文，不新增人物动机、事件或因果。
- 改动没有污染原文整理、分集文本或原文画本。
- 没有无依据改剧情。
- 与输入底稿差异可回溯。
- 后续动作明确：告一段落 / 进入录制版 / 待确认。
- 制作人确认状态明确。

### 4.8 录制版画本

- 角色颜色已渲染或有明确色标。
- 录制责任 / CV 按需显示。
- 同场景兼任提示存在或明确无风险。
- 前情提要位置正确。
- 待确认 / 补录记录存在。
- 可直接交给录制者使用。

### 4.9 试录校准

- 实际成品时长已记录。
- 片头、前情提要、片尾时长已扣除。
- 实际正文时长已计算。
- 校准朗读速度已计算。
- 偏差等级已判断。
- 后续集数是否调整有结论。
