## Excel 排版建议

| 设置 | 建议 |
|---|---|
| 工作表 | 按上述 5 张表拆分 sheet |
| 冻结窗格 | 每张表冻结首行；01_角色总表可冻结前 2-3 列 |
| 筛选 | 每张表开启筛选 |
| 自动换行 | 长文本列开启自动换行 |
| 颜色 | A 类深色、B 类中性色、C 类浅色；待确认项用警示色 |
| 条件格式 | 制作优先级、同场景兼任风险、试音优先级、置信度、状态 |
| 数据校验 | 角色档位、角色类型、风险等级、状态等字段设置下拉选项 |
