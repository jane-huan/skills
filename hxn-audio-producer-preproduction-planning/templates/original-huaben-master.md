# 《书名》原文画本母版 v0.6.9

> 用途：这是 AI / 制作人共同维护的“原文画本母版”。
> 原则：忠于原文，只做演播身份、前情提要、场景、音效、心声、系统音、群体声等结构标注；不绑定 CV；不做最终排版。
> 颜色规则：原文画本可以引用角色色标，但不负责最终彩色排版；颜色绑定角色，不绑定 CV。
> 后续：由本文件 +「角色颜色与录制责任表」派生成 Word/WPS 彩色录制版画本。

## 2. 原文画本正文

> 使用规则：
> - 前情提要：`【前情提要】内容`，默认放在片头之后、正文之前；若只用于时长计算、不进入画本正文，则不写在正文区。
> - 旁白：`【旁白】正文`
> - 角色对白：`【角色名（情绪/状态）】台词`
> - 心声：`【角色名｜心声（情绪/状态）】心声内容`
> - 系统音：`【系统音】系统提示内容`
> - 群体声：`【群体声｜处理方式（状态）】内容`，处理方式可为：多人同录 / 单人兼任 / 后期叠加 / 音效替代 / 待定。
> - 场景 / 音效：单独成行，可同一行并列：`【场景：...】【音效：...】`
> - 不在原文画本正文里写 CV 名、字体和最终排版样式。

## 4. 派生成录制版画本时的处理说明

派生 Word/WPS 录制版画本时再加入或渲染：

- 角色颜色；
- 角色样式；
- 按需显示录制责任人 / CV；
- 同场景 CV 兼任提示；
- 页眉页脚；
- 录制注意事项；
- 前情提要实际时长回填；
- 单 CV 角色工作稿（仅按需派生，不是默认产物）。

本 Markdown 母版保持干净，不直接承担最终排版。
